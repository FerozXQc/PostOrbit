**About PostOrbit**

PostOrbit is a sleek post scheduler designed to make content planning a breeze. Built with React, FastAPI, MySQL, and Redis, itâ€™s all about helping you schedule posts effortlessly and keep your workflow smooth.
**Installation**

    Clone the repo: git clone https://github.com/yourusername/postorbit.git
    Backend: cd backend, pip install -r requirements.txt, uvicorn main:app --reload
    Frontend: cd frontend, npm install, npm start
    Set up MySQL and Redis (check .env for config).

**Usage**

Hit http://localhost:3000, sign up, and start scheduling posts.
